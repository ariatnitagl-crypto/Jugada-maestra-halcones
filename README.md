Proyecto de Sistema de Información asistido por IA para la gestión del club Los Halcones del Barrio.
# ⚽ Proyecto: Jugada Maestra - Los Halcones del Barrio

Este proyecto es una solución tecnológica integral diseñada para automatizar la gestión administrativa y financiera de un club de fútbol amateur.

## 🛠️ Stack Tecnológico
**Backend:** Python con FastAPI.
**Frontend:** HTML5, JavaScript y Tailwind CSS.
**Base de Datos:** SQLite / SQL Relacional.
**IA Generativa:** NotebookLM y Google Gemini.

## 📂 Estructura del Repositorio
/backend: Servidor y lógica de negocio.
/frontend: Interfaz de usuario para Don Rafa.
/db: Scripts de creación de base de datos.
/docs: Informe ejecutivo, Bitácora de Prompts y Enlaces al Prototipo.

## 🚀 Cómo ejecutar
1. En la carpeta backend, instalar dependencias: pip install -r requirements.txt
2. Correr el servidor: uvicorn main:app --reload
3. Abrir frontend/index.html en el navegador.
